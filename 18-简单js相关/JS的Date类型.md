# JS
 
1 date类型及其转换
```
 var time = new Date();
time.getTime()
1567411847439

time.toDateString()
"Mon Sep 02 2019"

time.toGMTString()
"Mon, 02 Sep 2019 08:10:47 GMT"

time.toISOString()
"2019-09-02T08:10:47.439Z"

time.toJSON()
"2019-09-02T08:10:47.439Z"

time.toLocaleString()
"2019/9/2 下午4:10:47"

time.toLocaleDateString()
"2019/9/2"

time.toLocaleTimeString()
"下午4:10:47"

time.toString()
"Mon Sep 02 2019 16:10:47 GMT+0800 (中国标准时间)"

time.toTimeString()
"16:10:47 GMT+0800 (中国标准时间)"

time.toUTCString()
"Mon, 02 Sep 2019 08:10:47 GMT"
```
